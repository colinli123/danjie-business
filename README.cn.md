蛋解商业是一套综合性商业认知能力，萃取全球十位极具影响力商业领袖的核心智慧。
涵盖雷军顺势而为理念、巴菲特价值投资、马斯克第一性原理、彼得·蒂尔从0到1思维等内容，为**战略决策、投资分析、创业经营**提供可直接落地的思维框架。

## 核心特色
- **十大商业大师智慧**：深度拆解国内商界大佬、全球投资巨头、硅谷科技领军人物的思维模型
- **六大知识模块**：商业本质、思维模型、行业洞察、人生智慧、决策框架、经典案例
- **实战落地应用**：清晰指导每条商业原理的适用场景与使用方法

## 快速使用指南
1. 需要商业认知、商业决策建议时，启用本能力
